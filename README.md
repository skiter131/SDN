# SDN
Software defined networking for the Brazilian army

## Start

### Rodar docker-compose
```
make docker-comp
```

### Rodar docker-compose windows 

```
make docker-win
```

### Request API 

localhost:5002/apipatrulha

### Dependencias 

- Authorization Token
- JSON Body
- Method POST
